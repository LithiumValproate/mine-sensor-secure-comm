# 参考资料

## RFC / 标准文档

1. **RFC 5246** — The Transport Layer Security (TLS) Protocol Version 1.2. T. Dierks, E. Rescorla. IETF, August 2008.
   https://www.rfc-editor.org/info/rfc5246

2. **RFC 8446** — The Transport Layer Security (TLS) Protocol Version 1.3. E. Rescorla. IETF, August 2018.
   https://www.rfc-editor.org/info/rfc8446

3. **MQTT Version 3.1.1** — OASIS Standard. A. Banks, R. Gupta (eds.). October 2014.
   https://docs.oasis-open.org/mqtt/mqtt/v3.1.1/os/mqtt-v3.1.1-os.html

4. **NIST SP 800-38D** — Recommendation for Block Cipher Modes of Operation: Galois/Counter Mode (GCM) and GMAC. M. Dworkin. NIST, November 2007 (Revision announced 2024).
   https://csrc.nist.gov/publications/detail/sp/800-38d/final

5. **RFC 5869** — HMAC-based Extract-and-Expand Key Derivation Function (HKDF). H. Krawczyk. IETF, May 2010.
   https://www.rfc-editor.org/info/rfc5869
