**项目名称：** compNet - 矿井传感器安全通信系统  
**开发语言：** Python 3.11+  
**协议标准：** MQTT 3.1.1 + TLS 1.2  

---

## 1. 目的

- **理解MQTT协议与TLS加密通信的原理与实现细节**：深入掌握MQTT协议规范和TLS双向认证机制。

- **掌握网络编程技术**：熟练使用Python `paho-mqtt` 库进行MQTT客户端开发，实现传感器节点与地面中心的双向通信。

- **实现理论与实践结合**：将密码学理论（AES-GCM、数字证书、预共享密钥）应用于实际系统，培养网络安全问题分析与解决能力。

- **构建完整系统**：实现传感器数据采集、加密传输、安全验证、本地启动编排与Web监控展示的完整链路。

## 4. 系统设计与实现

### 4.1 系统架构

该系统为基于MQTT over TLS的矿井通信安全系统，采用三层架构实现井下数据的安全采集、传输与监控：

- **底层（传感器节点层）**：部署于采煤工作面、掘进面等场景的传感器节点，负责温度、瓦斯浓度采集，通过AES-GCM加密数据后，以MQTT客户端身份向中间层发送消息，并支持LWT遗嘱消息保障离线状态感知；
- **中间层（MQTT Broker层）**：由Mosquitto Broker承载，开放8883端口、运行MQTT协议，通过**TLS 1.2双向认证**建立安全通道，作为"枢纽"转发底层与顶层的加密数据；
- **顶层（地面中心服务器）**：依次完成"数据接收→解密→验证→告警"流程，并由本地启动器提供Web监控模块呈现实时状态、日志和告警。

### 4.2 核心代码实现

#### 4.2.1 应用层加密模块

**加密工具类**（[crypto_utils.py](src/mine_sensor_secure_comm/crypto_utils.py)）：

```python
def derive_aes128_key(psk_hex: str, sensor_id: str) -> bytes:
    """使用HKDF-SHA256从PSK派生每个传感器独立的AES-128密钥"""
    psk = bytes.fromhex(psk_hex)
    hkdf = HKDF(
        algorithm=hashes.SHA256(),
        length=KEY_SIZE,
        salt=sensor_id.encode('utf-8'),
        info=HKDF_INFO,
    )
    return hkdf.derive(psk)

def encrypt_payload(
        *,
        psk_hex: str,
        sensor_id: str,
        sensor_type: str,
        seq: int,
        timestamp_ms: int,
        plaintext: dict[str, Any],
        boot_random: bytes,
        version: int = 1,
) -> EncryptedPayload:
    """使用AES-128-GCM加密传感器读数"""
    key = derive_aes128_key(psk_hex, sensor_id)
    nonce = make_nonce(boot_random, seq)
    aad = build_aad(version, sensor_id, sensor_type, seq, timestamp_ms)
    encrypted = AESGCM(key).encrypt(nonce, canonical_json(plaintext), aad)
    ciphertext = encrypted[:-GCM_TAG_SIZE]
    tag = encrypted[-GCM_TAG_SIZE:]
    return EncryptedPayload(
        version=version,
        sensor_id=sensor_id,
        sensor_type=sensor_type,
        seq=seq,
        timestamp_ms=timestamp_ms,
        nonce=b64url_encode(nonce),
        ciphertext=b64url_encode(ciphertext),
        tag=b64url_encode(tag),
    )
```

**加密机制特点**：
- 使用HKDF-SHA256从主PSK派生每个传感器的独立密钥
- AES-GCM提供机密性和完整性保护
- AAD（附加认证数据）包含版本号、传感器ID、类型、序列号和时间戳
- Nonce由64位启动随机数+32位序列号组成，确保唯一性

def make_tls_client(
        *,
        client_id: str,
        ca_file: str,
        cert_file: str,
        key_file: str,
) -> mqtt.Client:
    """创建已配置mTLS的paho-mqtt客户端"""
    client = mqtt.Client(mqtt.CallbackAPIVersion.VERSION2, client_id=client_id)
    client.tls_set(
        ca_certs=ca_file,
        certfile=cert_file,
        keyfile=key_file,
        tls_version=ssl.PROTOCOL_TLS_CLIENT,
    )
    client.tls_insecure_set(False)
    return client

#### 4.2.3 防重放攻击机制

**序列号验证**（[replay_guard.py](src/mine_sensor_secure_comm/replay_guard.py)）：

```python
class ReplayGuard:
    """校验时间戳新鲜度和重复序列号"""
    
    def __init__(self, window_ms: int = 300_000) -> None:
        self.window_ms = window_ms  # 5分钟时间窗口
        self._states: dict[str, SensorReplayState] = {}
    
    def check(self, sensor_id: str, seq: int, timestamp_ms: int, receive_time_ms: int) -> str | None:
        # 时间戳新鲜度检查
        if abs(receive_time_ms - timestamp_ms) > self.window_ms:
            return 'timestamp_out_of_window'
        
        state = self._states.setdefault(sensor_id, SensorReplayState())
        self._prune(state, receive_time_ms)
        
        # 序列号重复检查
        if seq in state.recent_seq:
            return 'replay_detected'
        
        # 序列号回退检查
        if seq <= state.last_seq:
            return 'sequence_rollback'
        
        return None
```

**防重放策略**：
- 时间窗口验证：消息时间戳与接收时间差不超过5分钟
- 序列号重复检测：维护最近接收的序列号集合
- 序列号回退检测：拒绝序列号小于已接收最大序列号的消息

#### 4.2.4 地面中心处理核心

**消息处理逻辑**（[center_core.py](src/mine_sensor_secure_comm/center_core.py)）：

```python
class GroundCenterCore:
    """供MQTT回调和测试复用的纯校验核心"""
    
    def __init__(
            self,
            *,
            psk_map: dict[str, str],
            sensor_types: dict[str, str],
            thresholds: dict[str, dict[str, float]],
            replay_guard: ReplayGuard | None = None,
    ) -> None:
        self.psk_map = psk_map
        self.sensor_types = sensor_types
        self.thresholds = thresholds
        self.replay_guard = replay_guard or ReplayGuard()
    
    def process_data_message(
            self,
            payload: dict[str, Any],
            *,
            certificate_identity: str | None,
            receive_time_ms: int,
    ) -> ProcessResult:
        sensor_id = str(payload.get('sensor_id', ''))
        sensor_type = str(payload.get('sensor_type', ''))
        
        # 身份验证
        if sensor_id not in self.psk_map or sensor_id not in self.sensor_types:
            return self._reject('unknown_sensor', sensor_id, {'sensor_id': sensor_id})
        
        # 证书身份校验（当前默认运行链路通常拿不到对端证书CN，因此这里只在集成层显式提供时生效）
        if certificate_identity is not None and certificate_identity != sensor_id:
            return self._reject('identity_mismatch', sensor_id, {
                'certificate_identity': certificate_identity,
                'payload_sensor_id': sensor_id,
            })
        
        # 传感器类型校验
        if self.sensor_types[sensor_id] != sensor_type:
            return self._reject('identity_mismatch', sensor_id, {
                'expected_sensor_type': self.sensor_types[sensor_id],
                'payload_sensor_type': sensor_type,
            })
        
        # 重放检测
        replay_error = self.replay_guard.check(sensor_id, seq, timestamp_ms, receive_time_ms)
        if replay_error:
            return self._reject(replay_error, sensor_id, {'seq': seq, 'timestamp_ms': timestamp_ms})
        
        # AES-GCM解密
        plaintext = decrypt_payload(self.psk_map[sensor_id], payload)
        
        # 阈值告警检测
        value = plaintext.get('value')
        if isinstance(value, int | float):
            alert = threshold_alert(sensor_id, sensor_type, float(value), self.thresholds)
            if alert is not None:
                alerts.append(alert)
        
        return ProcessResult(accepted=True, plaintext=plaintext, alerts=alerts)
```

**处理流程**：
1. 传感器身份验证（PSK映射检查）
2. 证书身份校验（可选；默认运行链路通常由Broker负责接入层校验）
3. 传感器类型校验
4. 重放攻击检测（序列号+时间戳）
5. AES-GCM解密与完整性验证
6. 阈值告警检测

```python
def threshold_alert(
        sensor_id: str,
        sensor_type: str,
        value: float,
        thresholds: dict[str, dict[str, float]],
) -> Alert | None:
    sensor_threshold = thresholds.get(sensor_type, {})
    critical = sensor_threshold.get('critical')
    warning = sensor_threshold.get('warning')
    
    if critical is not None and value >= critical:
        return Alert(
            code=f"{sensor_type}_threshold_exceeded",
            sensor_id=sensor_id,
            severity='high',
            message=f"{sensor_type} critical threshold exceeded",
            details={'value': value, 'threshold': critical},
        )
    
    if warning is not None and value >= warning:
        return Alert(
            code=f"{sensor_type}_threshold_warning",
            sensor_id=sensor_id,
            severity='medium',
            message=f"{sensor_type} warning threshold exceeded",
            details={'value': value, 'threshold': warning},
        )
    
    return None
```

### 4.3 安全机制总结

| 安全层 | 机制 | 实现方式 |
|--------|------|----------|
| **传输层** | TLS 1.2双向认证 | 客户端/服务端证书验证 |
| **应用层** | AES-128-GCM加密 | 认证加密（AEAD） |
| **密钥管理** | HKDF-SHA256密钥派生 | 每个传感器独立密钥 |
| **身份认证** | PSK + mTLS接入控制 | PSK映射 + Broker证书接入校验 |
| **防重放** | 序列号+时间戳 | 5分钟时间窗口 |
| **完整性** | GCM认证标签 | AAD保护元数据 |
| **断线检测** | LWT遗嘱消息 | MQTT KeepAlive机制 |
